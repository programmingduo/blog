# 开发手册
Python安装后自带了一个叫做pydoc.py的文件。使用命令行在Python的安装目录下lib路径当中，执行命令

~~~~~~~
python pydoc.py -p 8787
~~~~~~~
之后在浏览器中输入url    

~~~~
localhost:8787
~~~~

便可阅读该文档了。
